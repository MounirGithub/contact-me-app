# Decision Log — Portfolio Brief

## 2026-06-02

### Ordre des phases
- **Décision** : Refacto → Bugs/UX → Contenu
- **Raison** : Corriger sur une base fragile = sparadrap. La refacto pose le socle pour que tout le reste soit chirurgical et durable.

### Migration CRA → Vite
- **Décision** : Adopter Vite
- **Raison** : CRA n'est plus maintenu (react-scripts 5.0.1 date de 2022). Vite est 10x plus rapide en build, HMR instantané, écosystème vivant.

### Styling : CSS Modules
- **Décision** : CSS Modules (.module.css par composant), pas Styled Components ni Tailwind
- **Raison** : Scoping natif, zéro runtime, zéro paradigm shift par rapport au CSS actuel. Facile à migrer depuis le CSS monolithique.

### Smooth scroll : Lenis
- **Décision** : Utiliser Lenis (studio-freight/lenis)
- **Raison** : Léger, React-friendly, utilisé par les sites type caoenergy. Pas de locomotive-scroll (plus lourd, moins maintenu).

### Animations : Framer Motion
- **Décision** : Framer Motion pour les animations d'entrée et transitions
- **Raison** : Standard React, API déclarative, remplace le bricolage IntersectionObserver + keyframes. react-intersection-observer reste possible en interne pour le trigger mais l'animation est déléguée à FM.

### Pas de TypeScript
- **Décision** : Rester en JavaScript
- **Raison** : Overhead de migration sans bénéfice user visible. Portfolio solo, pas une app d'équipe. TS pourra être adopté plus tard si le projet grandit (blog CMS, etc.).

### Structure dossiers
- **Décision** : `src/components/ComponentName/index.jsx` + `ComponentName.module.css`
- **Raison** : Un dossier par composant = localité, facile à naviguer, imports propres.

### Thème visuel
- **Décision** : Conserver sky-to-ocean tel quel
- **Raison** : Le design est validé par l'utilisateur. Pas de changement de palette, typo ou concept visuel.

### Inspiration UX
- **Décision** : caoenergy.com comme référence de fluidité (pas de copie de code/DOM)
- **Raison** : L'utilisateur vise le même feeling : mouvement naturel, profondeur, scroll sans saccade.
